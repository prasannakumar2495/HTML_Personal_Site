        The full form of HTML is Hyper Text Markup Language. HTML is basically the skeleton of a website. HTML will be
        used to decide the structure of a website using a series of elements. We can create a website just wait just by
        using HTML but we cannot create a website just by using CSS for JavaScript. While creating a website HTML is
        mandatory.

There are so many tag elements in HTML (most of the HTML tag elements have closing except few). Few of them are as follows:
  • h1 - This is the heading tag element, this is the first one in the series of heading tag elements and size of this tag element is the biggest of all the heading tags elements.
  • br - This element is used to provide space Between different tag elements. This tiger man does not have a closing.
  • strong - Any word or sentence that is written in between this tag element will be displayed in bold showcasing that this is an important sentence.
        The tag elements also consist of many attributes that are written within them which are usually defined use to
        define the style and various other properties.
        While writing the HTML code the programmer can write comments in between the "but this comment will not be
        displayed in the website.
